## Веб-интерфейс LuCI
- URL: http://192.168.2.200
- Логин: root / пароль пустой (сменить после входа)

## SSH
- ssh root@192.168.2.200

## Физические пользователи (свитч на enp1s0 / vmbr1)
- Подключить свитч к enp1s0 на сервере Proxmox
- Устройства получат IP по DHCP: 10.10.10.100-200
- Шлюз: 10.10.10.1 (OpenWrt LAN)
- DNS: 10.10.10.1

## Доступ из 192.168.2.0/24 в сеть 10.10.10.0/24
- Добавить маршрут на клиенте:
  Linux/Mac:  ip route add 10.10.10.0/24 via 192.168.2.200
  Windows:    route add 10.10.10.0 mask 255.255.255.0 192.168.2.200
- Или добавить статический маршрут на основном роутере 192.168.2.1

## Proxmox serial console (если нет сети)
- qm terminal 200
