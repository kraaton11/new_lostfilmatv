# LostFilm New TV

> **Отказ от ответственности**
>
> Это неофициальный Android TV клиент для `lostfilm.today`. Проект не связан с LostFilm.TV или правообладателями контента.

Android TV приложение для просмотра новых релизов, фильмов, расписания и избранного LostFilm в интерфейсе, оптимизированном для управления пультом.

## Возможности

- Лента новых релизов и отдельный раздел фильмов с `lostfilm.today`
- Поиск по каталогу
- Расписание релизов
- Персональное избранное после входа в аккаунт LostFilm
- QR-авторизация без ввода логина и пароля на телевизоре
- Карточка релиза с постером, описанием, рейтингом TMDB и торрент-раздачами
- Обзор сериала или фильма
- Гид по сезонам и сериям с отметками просмотра
- Запуск воспроизведения через TorrServe
- Канал Android TV Home Channel с фоновым обновлением
- Проверка обновлений приложения через GitHub Releases
- Локальный кеш данных для более стабильной работы

## Требования

- **Устройство:** Android TV / Google TV с Android 8.0+ (API 26+)
- **TorrServe:** Обязателен для воспроизведения видео ([скачать TorrServe](https://github.com/YouROK/TorrServe/releases))
- **Аккаунт LostFilm:** Не обязателен для просмотра новых релизов и фильмов, но нужен для избранного и персональных функций

## Установка

1. **Скачайте APK**
   - Откройте страницу [Releases](../../releases/latest)
   - Скачайте последнюю версию APK

2. **Установите TorrServe**
   - Скачайте [TorrServe для Android](https://github.com/YouROK/TorrServe/releases)
   - Установите на ваше Android TV устройство
   - Запустите TorrServe (работает на `http://127.0.0.1:8090`)

3. **Установите LostFilm New TV**
   - Перенесите APK на Android TV через USB или сетевой файловый менеджер
   - Откройте файловый менеджер на телевизоре и установите APK
   - Или используйте ADB: `adb install app-release.apk`

4. **Запустите приложение**
   - Найдите LostFilm New TV в списке приложений
   - При необходимости войдите в аккаунт LostFilm через QR-код

## Часто задаваемые вопросы

### TorrServe не подключается

Убедитесь, что:
- TorrServe установлен и запущен на вашем устройстве
- TorrServe работает на порту `8090` (проверьте в настройках TorrServe)
- В настройках приложения указан правильный адрес TorrServe (по умолчанию `http://127.0.0.1:8090`)

### Видео не воспроизводится

- Проверьте подключение к интернету
- Убедитесь, что TorrServe активен и доступен
- Попробуйте выбрать другое качество или озвучку

### QR-авторизация не работает

- Убедитесь, что телефон и телевизор в одной сети
- Попробуйте отсканировать QR-код камерой телефона повторно
- Проверьте, что браузер на телефоне открыл страницу авторизации

### APK не устанавливается

- Разрешите установку из неизвестных источников в настройках Android TV
- Проверьте, что скачали правильную версию APK для вашей архитектуры
- Попробуйте установить через ADB
